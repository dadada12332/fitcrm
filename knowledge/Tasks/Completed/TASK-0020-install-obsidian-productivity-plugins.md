# Install Obsidian productivity plugins

Installed and enabled Dataview, Tasks, Periodic Notes, Templater and Style Settings from repositories listed in the official Obsidian community plugin registry. The same plugin set is present in the repository vault and the synced personal vault.

Periodic Notes is configured for `Weekly/YYYY-[W]WW.md` with `Templates/Weekly Review Template.md`. Templater uses `Templates`, while automatic file triggers, startup templates and system commands remain disabled. Templater `2.20.6` is pinned because newer versions require Obsidian `1.13`; the installed application is `1.12.7`.

Obsidian verification showed all five plugins in the active third-party settings list. Templater and Periodic Notes also registered their ribbon actions after a full restart.
