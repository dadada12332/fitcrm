# Recommended Obsidian Plugins

Ни один плагин не обязателен: Markdown, ссылки и скрипты работают без них.

## Текущая настройка

- `Kanban` включён. `05 Kanban.md` генерируется как нативная доска из статусов task-файлов.
- `Calendar` включён, неделя начинается с понедельника. Дни связаны с `Daily/YYYY-MM-DD.md` и шаблоном `Templates/Daily Note Template.md`.
- `Obsidian Git` синхронизирует отдельный vault-репозиторий между компьютерами.
- `Local REST API` оставлен включённым для локальной автоматизации.
- `Dataview`, `Tasks`, `Periodic Notes`, `Templater` и `Style Settings` установлены и включены.
- `Periodic Notes` открывает `Weekly/YYYY-[W]WW.md` по шаблону `Templates/Weekly Review Template.md`.
- `Templater` использует папку `Templates`; автоматический запуск и системные команды выключены.

Task-файлы в `Tasks/` остаются источником истины. Ручное перемещение карточек на доске может быть перезаписано следующим `npm run docs:sync`; статус задачи нужно менять в её frontmatter.

| Plugin         | Зачем                                 | Обязателен | Без него                                         |
| -------------- | ------------------------------------- | ---------- | ------------------------------------------------ |
| Kanban         | Визуальная доска `05 Kanban.md`       | Нет        | Останется обычная Markdown-доска                 |
| Templater      | Ручная вставка шаблонов               | Нет        | Используются npm-команды или копирование шаблона |
| Dataview       | Дополнительные выборки по frontmatter | Нет        | Dashboard остаётся статичным + AUTO-секции       |
| Obsidian Git   | Pull/push из приложения               | Нет        | Используется обычный Git                         |
| Calendar       | Навигация по Daily                    | Нет        | Daily доступны по папке и ссылкам                |
| Periodic Notes | Weekly/Daily UX                       | Нет        | Файлы создаются npm-командами                    |
| Tasks          | Расширенные task queries              | Нет        | Kanban и task-файлы читаются напрямую            |

Плагины не устанавливаются автоматически и их локальные файлы игнорируются Git.

## Дополнительно установлено

1. `Dataview 0.5.68` — живые таблицы задач, решений, инцидентов и просроченных пунктов прямо на Dashboard.
2. `Tasks 8.2.2` — повторяющиеся задачи, сроки, напоминания и удобные выборки незавершённого.
3. `Periodic Notes 1.0.0-beta.3` — недельные обзоры, связанные с Calendar.
4. `Templater 2.20.6` — умные формы для новых task, ADR, incident и meeting notes с автодатами. Зафиксирована последняя версия, совместимая с Obsidian 1.12.7.
5. `Style Settings 1.0.9` — визуальная настройка темы и CSS-сниппетов без ручного редактирования стилей.
