# FitCRM Knowledge Vault

Операционная память проекта, читаемая в Obsidian, GitHub, VS Code и AI-инструментах без обязательных плагинов.

Начальная точка для человека — [[00 Dashboard]]. Начальная точка для AI — [[10 AI Handoff]].

## Правила

- Код, Git и миграции важнее заметок.
- Ручной текст не изменяется автоматизацией.
- Скрипты могут менять только секции `AUTO:START` / `AUTO:END`.
- `.obsidian/` не хранится в Git: каждый компьютер может иметь собственный layout и плагины.
- Секреты и персональные данные в Vault запрещены.
