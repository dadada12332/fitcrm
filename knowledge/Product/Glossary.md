# Glossary

| Термин | Значение |
|---|---|
| club | Фитнес-клуб, основная единица данных и биллинга |
| tenant | Изолированный контур клуба; в FitCRM соответствует `club_id` |
| member/client | Клиент фитнес-клуба; в коде и БД используется `client` |
| membership | Шаблон/тип абонемента, который клуб продаёт клиентам |
| subscription | Проданный клиенту экземпляр абонемента со сроком и статусом |
| payment | Платёж клиента клубу; не путать с оплатой тарифа FitCRM |
| visit | Посещение/check-in клиента |
| platform admin | Оператор всей SaaS-платформы, не сотрудник одного клуба |
| club admin | Сотрудник конкретного клуба с административными правами |
| RPC | Postgres-функция, вызываемая через Supabase API |
| RLS | Row Level Security, tenant-изоляция строк в Postgres |
| white label | Брендирование CRM под конкретный клуб |
| trial | Пробный период тарифа клуба, сейчас 14 дней |
| plan | Тариф FitCRM для клуба |
